# Primeiro Projeto Java: Mercado 🛒

Primeiro projeto desenvolvido em Java, marcando o início da minha jornada na programação. O sistema simula um mercado via console, utilizando arrays e lógica estruturada para gerenciar catálogo, estoque, carrinho de compras com adição/remoção de itens e aplicação de cupons de desconto.

### ✨ Funcionalidades

* **Catálogo Dinâmico:** Visualização de itens disponíveis com preços formatados.
* **Gerenciamento de Inventário:** Verificação automática de disponibilidade de estoque antes da adição ao carrinho.
* **Manipulação de Carrinho:** Adição e remoção de itens com atualização em tempo real dos subtotais.
* **Sistema de Descontos:** Aplicação de cupons (ex: `mercado10`) para cálculo de preços promocionais.
* **Recibo Detalhado:** Geração de um resumo final com itens, quantidades, valores e o total geral da compra.
